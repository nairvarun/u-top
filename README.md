## Setup

### Step 1:
Install Go from official website (https://go.dev/)

### Step 2:
Clone this repo
```git clone https://github.com/nairvarun/u-top.git```

### Step 3:
run ```go mod tidy```

### Step 4:
Initialise MySQL using ```db/setup.sql```

### Step 5:
run ```go run .```
